Local

POST Signup

http://localhost:10450/signup

BODY -- JSON DATA username, password, confirmpassword


POST Authenticate

http://localhost:10450/authentictae

BODY -- JSON DATA username, password


POST Login

http://localhost:10450/login

BODY -- JSON DATA username, password

HEADERS --- token


GET Product

http://localhost:10450/api/product?limit=2&category=camera

PARAMS -- limit, category

HEADERS --- token


POST Product

http://localhost:10450/api/product

BODY -- JSON DATA refer document

HEADERS --- token


PUT Product

http://localhost:10450/api/product?id=[string]

PARAMS -- id

HEADERS --- token


DELETE Product

http://localhost:10450/api/product?id=[string]

OR

http://localhost:10450/api/product?name=[string]&category[string]

OR

http://localhost:10450/api/product?name=[string]

OR

http://localhost:10450/api/product?category[string]

PARAMS -- id, name, category

HEADERS --- token


GET Product

http://localhost:10450/logout

HEADERS --- token
